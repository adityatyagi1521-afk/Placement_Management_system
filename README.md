# 🚀 Placement Management System (C++)

## 📌 Project Description

This is a **Placement Management System** developed using **C++**.
The system helps manage student data, company job postings, and job applications in a simple console-based interface.

It simulates a real-world college placement process where students can apply for jobs and companies can manage requirements.

---

## 🎯 Features

* 👨‍🎓 Add Student Details (Name, Roll No, Course, College, Year)
* 🏢 Add Company Details (Name, Job Post, Salary, Eligibility)
* 📄 Apply for Job
* 📋 View All Applications
* 💾 File Handling (Data stored in `.dat` files)
